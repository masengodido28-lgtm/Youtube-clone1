# YouTube Clone

A responsive YouTube-inspired web application built with HTML, CSS, and JavaScript. This project recreates the core layout and user interface of YouTube while helping me practice front-end development skills.

## Features

- Responsive navigation bar
- Sidebar menu
- Video grid layout
- Video cards with thumbnails
- Channel profile icons
- Search bar
- Category filter buttons
- Mobile-friendly design
- Hover effects and interactive UI elements

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Google Material Symbols

## Getting Started

1. Clone the repository:

```bash
git clone <repository-url>
```

2. Navigate to the project folder:

```bash
cd youtube-clone
```

3. Open `index.html` in your browser.
